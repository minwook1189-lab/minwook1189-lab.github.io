## 📋 시연 순서 및 스크립트

### 시연 1: 고객용 대시보드 (10분) ⭐ 하이라이트

**스토리라인:**
> "고객사에서 본인들 물량 진행 상황을 실시간으로 보고 싶다고 합니다.
> 개발팀에 요청하면 몇 달 걸릴 텐데, AI로 프로토타입을 만들어봤습니다."

**시연 순서:**
1. Cursor 열고 빈 HTML 파일 생성
2. Claude에게 요청: *"반도체 파운드리 고객 포털 대시보드를 만들어줘. 생산 현황, Lot 진행 상태, 수율 정보를 보여주는 페이지"*
3. 생성된 코드 → 브라우저에서 바로 확인
4. **즉석 수정 요청**: *"차트 색상을 회사 CI로 바꿔줘"* 또는 *"고객사 로고 추가해줘"*

**핵심 포인트:**
- 코딩 없이 아이디어만으로 프로토타입 가능
- 기획 → 시각화까지 몇 분
- 개발 요청 전 컨셉 검증에 활용

**준비된 파일로 보여주기:**
- [01_customer_dashboard.html](01_customer_dashboard.html)을 브라우저에서 열어서 완성본 시연

### 시연 3: AI 활용 학습 (5분)

**스토리라인:**
> "직원 역량 개발에 AI를 어떻게 활용할 수 있는지 보여드리겠습니다."

**시연 순서:**
1. 파이썬 학습 예시: *"for 반복문이 뭔지 쉽게 설명해줘"*
2. 코드 작성 요청: *"매출 데이터에서 평균 이상만 필터링하는 코드"*
3. 에러 해결: 일부러 에러 있는 코드 보여주고 *"이거 왜 에러나?"*
4. 영어 학습 예시: *"납기 지연 사과 메일 작성해줘"*

**핵심 포인트:**
- 1:1 맞춤 튜터링 가능
- 실무 맥락에 맞춘 학습
- 즉각적인 피드백

**참고 자료:**
- [02_python_learning_curriculum.md](02_python_learning_curriculum.md)
- [03_english_learning_curriculum.md](03_english_learning_curriculum.md)

### 시연 4: 메일 자동화 (5분)

**스토리라인:**
> "매일 아침 생산 현황을 정리해서 메일 보내는 작업, 자동화할 수 있습니다."

**시연 순서:**
1. 요구사항 설명: *"매일 생산 데이터를 예쁜 형태로 정리해서 Outlook으로 보내고 싶어"*
2. 파이썬 스크립트 생성 과정 설명
3. 실행해서 `email_preview.html` 열어보기
4. *"수율이 93% 미만일 때만 긴급 알림 보내게 해줘"* 조건 추가

**핵심 포인트:**
- 반복 업무 자동화
- 비개발자도 AI 도움으로 스크립트 작성 가능
- 점진적 확장 (알림 조건, 데이터 소스 등)

**참고 자료:**
- [04_daily_report_email.py](04_daily_report_email.py)

## 💡 시연 팁

### Do's ✅
- **실시간으로 보여주기**: 미리 만든 것 아닌 걸 강조
- **실패도 보여주기**: 100% 완벽하지 않음, 사람의 검토 필요
- **즉석 요청 받기**: "뭐 바꿔볼까요?" 관객 참여 유도
- **속도 강조**: "원래라면 며칠 걸릴 작업"

### Don'ts ❌
- 너무 기술적인 설명 피하기
- "AI가 다 해줍니다" 과장 금지
- 민감한 회사 데이터 사용 금지

## 🎤 예상 질문 & 답변

**Q: 보안은 괜찮은가?**
> 회사 데이터를 외부 AI에 직접 넣지 않고, 구조/형식만 참고합니다.
> 실제 데이터는 사내 시스템에서 처리하고, AI는 코드 작성만 도와줍니다.

**Q: 개발자가 필요 없어지나?**
> 프로토타입/POC는 빨라지지만, 실제 운영 시스템은 여전히 개발자 필요.
> 오히려 개발자의 생산성이 크게 향상됩니다.

**Q: 비용은?**
> Claude Pro 월 $20, Cursor Pro 월 $20
> 기업용 라이선스도 있음

**Q: 어디서 배우나?**
> 직접 써보는 게 최고의 학습. 오늘 만든 커리큘럼 참고.
