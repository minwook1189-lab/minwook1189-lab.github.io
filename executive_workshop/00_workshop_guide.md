### 시연 1: 고객용 대시보드 (10분) ⭐ 하이라이트

**스토리라인:**
> "고객사에서 본인들 물량 진행 상황을 실시간으로 보고 싶다고 합니다.
> 개발팀에 요청하면 몇 달 걸릴 텐데, AI로 지금 바로 만들어보겠습니다."

**시연 순서:**

1️⃣ **빈 폴더에서 시작** (Cursor에서)
   - 새 폴더 `demo_live/` 생성
   - Cursor로 열기
   - (선택) 05_demo_site_spec.md 파일을 폴더에 복사

2️⃣ **AI에게 웹사이트 생성 요청** (라이브로)
   - 아래 초기 프롬프트 중 택 1

3️⃣ **결과 확인**
   - 브라우저에서 index.html 열기
   - 페이지 전환, 차트 확인

4️⃣ **즉석 수정 요청** (라이브로)
   - 아래 수정 시나리오 중 택 1~2개 실행

**📝 초기 프롬프트 (택 1)**

| 버전 | 프롬프트 | 비고 |
|------|----------|------|
| A. 스펙 참조 | *"05_demo_site_spec.md 파일을 읽고 이 스펙대로 반도체 파운드리 고객 포털 웹사이트를 만들어줘"* | ⭐ 안정적 |
| B. 자연어 | *"반도체 파운드리 회사의 고객 포털 웹사이트를 만들어줘. 고객사가 자기 물량의 생산 현황을 보는 대시보드야. 홈, 생산현황, 품질리포트, 문의하기 4페이지로. Chart.js 사용해줘."* | ⭐ 인상적 |
| C. 간단 | *"반도체 파운드리 고객 포털 대시보드 만들어줘. 4페이지, Chart.js 차트 포함."* | 빠름 |

> 자세한 프롬프트 가이드: [06_demo_initial_prompt.md](06_demo_initial_prompt.md)

**📝 즉석 수정 시연 시나리오 (생성 후 택 1~2개)**

| 요청 | 예시 프롬프트 | 난이도 |
|------|---------------|--------|
| 회사명/로고 변경 | *"회사명을 SK키파운드리로 바꾸고, 로고도 우리 회사 홈페이지에서 가져와서 적용해줘"* | ⭐ 추천 |
| 드롭다운 작동 | *"월별 생산량 추이에서 '최근 6개월'을 '최근 12개월'로 바꾸면 차트가 바뀌게 해줘"* | ⭐ 추천 |
| 사용자명 변경 | *"로그인한 사용자 이름을 '김민욱'으로 바꿔줘"* | 쉬움 |
| 색상 변경 | *"차트 색상을 SK 브랜드 컬러(빨간색 계열)로 바꿔줘"* | 보통 |
| 품질차트 구현 | *"품질 리포트 페이지에 수율 트렌드 차트 추가해줘"* | ⭐ 추천 |

**핵심 포인트:**
- **처음부터 생성**: 빈 폴더에서 웹사이트가 만들어지는 과정 시연
- 코딩 없이 자연어 요청만으로 즉각 생성/수정 가능
- 기획자/현업이 직접 프로토타입 생성 가능
- 개발팀 요청 전에 컨셉 검증 및 피드백 반영

**참고 파일:**

- [06_demo_initial_prompt.md](06_demo_initial_prompt.md) - 초기 프롬프트 가이드

### 시연 2: 데이터 시각화 (5분)

**스토리라인:**
> "현업에서 엑셀로 관리하는 데이터가 있죠? 이걸 웹 차트로 시각화해보겠습니다."

**시연 순서:**

1️⃣ **Excel 파일 준비**
   - `sample_production_data.csv` 파일을 demo_live/ 폴더에 복사
   - (또는 시연 1에서 만든 웹사이트 폴더에 복사)

2️⃣ **AI에게 시각화 요청**
   - *"이 데이터로 막대 차트 만들어줘"*

3️⃣ **결과 확인**
   - 브라우저에서 생성된 차트 확인

4️⃣ **추가 요청 (선택)**
   - 다른 차트 유형으로 변경
   - 스타일 수정

**📝 시각화 프롬프트 시나리오**

| 요청 | 예시 프롬프트 | 비고 |
|------|---------------|------|
| 기본 막대 차트 | *"이 데이터로 막대 차트 만들어줘"* | ⭐ 메인 |
| 라인 차트 추가 | *"수율 데이터를 라인 차트로 추가해줘"* | 추가 시연 |
| 차트 스타일 | *"차트 색상을 파란색 계열로 바꿔줘"* | 쉬움 |
| 복합 차트 | *"생산량은 막대, 수율은 라인으로 같이 보여줘"* | 고급 |

**📊 샘플 데이터 구조** (sample_production_data.csv)

| 월 | 생산량 | 출하량 | 목표 | 수율 |
|----|--------|--------|------|------|
| 1월 | 4,500 | 4,200 | 4,000 | 94.2% |
| 2월 | 4,800 | 4,500 | 4,200 | 93.8% |
| ... | ... | ... | ... | ... |
| 12월 | 7,500 | 7,200 | 7,000 | 97.1% |

**핵심 포인트:**
- 엑셀 데이터를 웹 차트로 즉시 변환
- 비개발자도 데이터 시각화 가능
- 다양한 차트 유형 요청 가능

**참고 파일:**
- [sample_production_data.csv](sample_production_data.csv) - 샘플 생산 데이터

### 시연 3: AI 활용 학습 (5분)

**스토리라인:**
> "직원 역량 개발에 AI를 어떻게 활용할 수 있는지 보여드리겠습니다."

**시연 순서:**
1. 파이썬 학습 예시: *"for 반복문이 뭔지 쉽게 설명해줘"*

3. 에러 해결: 일부러 에러 있는 코드 보여주고 *"이거 왜 에러나?"*
4. 영어 학습 예시: *"납기 지연 사과 메일 작성해줘"*

**핵심 포인트:**
- 1:1 맞춤 튜터링 가능
- 실무 맥락에 맞춘 학습
- 즉각적인 피드백

**스토리라인:**

**핵심 포인트:**
- 반복 업무 자동화
- 비개발자도 AI 도움으로 스크립트 작성 가능

## 💡 시연 팁

### Do's ✅
- **실시간으로 보여주기**: 미리 만든 것 아닌 걸 강조
- **실패도 보여주기**: 100% 완벽하지 않음, 사람의 검토 필요
- **즉석 요청 받기**: "뭐 바꿔볼까요?" 관객 참여 유도
- **속도 강조**: "원래라면 며칠 걸릴 작업"

### Don'ts ❌
- 너무 기술적인 설명 피하기
- "AI가 다 해줍니다" 과장 금지
- 민감한 회사 데이터 사용 금지

## 🎤 예상 질문 & 답변

**Q: 보안은 괜찮은가?**
> 회사 데이터를 외부 AI에 직접 넣지 않고, 구조/형식만 참고합니다.
> 실제 데이터는 사내 시스템에서 처리하고, AI는 코드 작성만 도와줍니다.

**Q: 개발자가 필요 없어지나?**
> 프로토타입/POC는 빨라지지만, 실제 운영 시스템은 여전히 개발자 필요.
> 오히려 개발자의 생산성이 크게 향상됩니다.

**Q: 비용은?**
> Claude Pro 월 $20, Cursor Pro 월 $20
> 기업용 라이선스도 있음

**Q: 어디서 배우나?**
> 직접 써보는 게 최고의 학습. 오늘 만든 커리큘럼 참고.
