# 반도체 파운드리 고객 포털 - 웹사이트 스펙

---

## ⚡ 요약 (Quick Summary)

> **한 줄 설명:** 반도체 파운드리 고객사가 자사 물량의 생산 현황을 실시간으로 확인하는 대시보드

### 핵심 요구사항
- **4개 페이지**: 홈(대시보드), 생산현황, 품질리포트, 문의하기
- **기술**: HTML/CSS/JS만 사용, Chart.js로 차트 구현
- **디자인**: 파란색 메인 컬러, 깔끔하고 전문적인 UI
- **언어**: 한글

### 필수 포함 요소
| 페이지 | 핵심 요소 |
|--------|-----------|
| 홈 | 4개 지표 카드, 월별 생산량 차트(Chart.js), Lot 진행 테이블 |
| 생산현황 | Lot 목록 테이블, 진행률 바, 상태 뱃지 |
| 품질리포트 | 수율/품질 지표 카드, 품질 테이블 |
| 문의하기 | 문의 폼, 담당자 연락처 |

### 공통 요소
- 상단: 네비게이션 바 (로고 + 메뉴 + 알림 + 사용자)
- 하단: 푸터 (저작권 + 연락처)
- 회사명: SemiFab / 고객: Samsung Electronics

- **반응형 웹** - 데스크톱/태블릿/모바일 지원

## 페이지별 상세 스펙

### 1. index.html (메인 대시보드)

#### 1-1. 환영 섹션
```
안녕하세요, Samsung Electronics님
오늘의 생산 현황과 주요 알림을 확인하세요

[현재 날짜] [현재 시간 - 실시간 업데이트]
```

#### 1-2. 알림 배너
```
📢 LOT-2024-0891 최종 테스트 완료 - 내일 출하 예정입니다 [×닫기]
```

#### 1-3. 핵심 지표 카드 (4개 가로 배치)

| 아이콘 | 수치 | 라벨 | 변화 |
|--------|------|------|------|
| 📦 파랑 | 127 | 진행 중 Lot | +12% |
| 📈 초록 | 94.7% | 평균 수율 | +1.2% |
| 🚚 보라 | 45 | 이번 달 출하 예정 | 예정대로 |
| ⏱ 주황 | 18.5일 | 평균 Cycle Time | -2.3일 |

#### 1-4. 차트 & 알림 영역 (2열 그리드)

**좌측: 월별 생산량 차트**
- 드롭다운: 최근 6개월 / 최근 12개월
- Line Chart (Chart.js)
- 데이터셋 2개: 생산량(파랑), 출하량(초록)
- X축: 8월, 9월, 10월, 11월, 12월, 1월
- Y축: Lot 수량

**우측: 최근 알림 카드**
- 알림 목록 (4개 항목)
- 각 알림: 아이콘 + 내용 + 시간

#### 1-5. 주요 진행 Lot 테이블

| Lot ID | 제품 | 수량 | 현재 공정 | 진행률 | 상태 | 예상 완료 |
|--------|------|------|-----------|--------|------|-----------|
| LOT-2024-0892 | 5nm AP Chip | 25 wafers | Lithography | 67% | FAB | 01/28 |
| LOT-2024-0891 | 7nm GPU | 30 wafers | Final Test | 92% | TEST | 01/25 |
| LOT-2024-0888 | 5nm Modem | 20 wafers | Packaging | 100% | DONE | 01/24 |

- 진행률: 프로그레스바로 시각화
- 상태: 컬러 뱃지 (FAB=파랑, TEST=주황, DONE=초록)

#### 1-6. 빠른 링크 (3개 카드)
- 생산 현황 → production.html
- 품질 리포트 → quality.html
- 담당자 문의 → contact.html

### 컬러 팔레트

```css
/* 메인 컬러 */
--primary: #0066cc;        /* 파랑 - 주 컬러 */
--primary-dark: #004999;
--primary-light: #e6f0ff;

/* 상태 컬러 */
--success: #10b981;        /* 초록 */
--warning: #f59e0b;        /* 주황 */
--danger: #ef4444;         /* 빨강 */
--info: #3b82f6;           /* 파랑 */

/* 중성 컬러 */
--dark: #1a1a2e;
--gray-800: #1f2937;
--gray-600: #4b5563;
--gray-400: #9ca3af;
--gray-200: #e5e7eb;
--gray-100: #f3f4f6;
--white: #ffffff;
```

### 타이포그래피
- 폰트: 'Segoe UI', -apple-system, BlinkMacSystemFont, sans-serif
- 한글 지원

### 컴포넌트 스타일

**카드**
- 배경: 흰색
- 테두리: 없음
- 그림자: 은은한 박스섀도우
- 모서리: 12px 라운드

**버튼**
- Primary: 파란 배경, 흰 텍스트
- Secondary: 회색 배경, 어두운 텍스트
- 모서리: 8px 라운드

**테이블**
- 헤더: 회색 배경, 대문자
- 행 호버: 연한 회색 배경
- 테두리: 1px 회색 하단선

**상태 뱃지**
- FAB: 파란 배경/텍스트
- TEST: 주황 배경/텍스트
- DONE: 초록 배경/텍스트

**진행률 바**
- 높이: 20px
- 배경: 회색
- 채움: 파랑 (100%일 때 초록)
- 텍스트: 우측에 퍼센트 표시
