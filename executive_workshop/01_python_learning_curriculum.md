## 🤖 AI와 상호작용하며 배우는 방법

### 1. 개념 이해하기
```
사용자: "for 반복문이 뭐야? 쉽게 설명해줘"

AI: for 반복문은 "같은 작업을 여러 번 반복"할 때 사용합니다.

    예를 들어, 출석 체크를 한다고 생각해보세요:
    - "김철수!" → "네!"
    - "이영희!" → "네!"
    - "박민수!" → "네!"

    코드로 보면:
    students = ["김철수", "이영희", "박민수"]
    for name in students:
        print(f"{name}!")
```

### 2. 코드 작성 요청하기
```
사용자: "직원 이름과 급여가 있는 딕셔너리에서
       급여가 500만원 이상인 사람만 출력하는 코드 만들어줘"

AI: employees = {
        "김철수": 450,
        "이영희": 520,
        "박민수": 600,
        "정지은": 480
    }

    for name, salary in employees.items():
        if salary >= 500:
            print(f"{name}: {salary}만원")

    # 출력:
    # 이영희: 520만원
    # 박민수: 600만원
```

### 3. 에러 해결하기
```
사용자: "이 코드 실행하면 에러나는데 뭐가 문제야?

       for i in range(10)
           print(i)"

AI: 콜론(:)이 빠졌습니다! for문 끝에 콜론을 붙여야 해요.

    수정된 코드:
    for i in range(10):  # ← 여기 콜론 추가
        print(i)
```

### 4. 코드 설명 요청하기
```
사용자: "이 코드가 뭘 하는 건지 설명해줘

       df = df[df['매출'] > df['매출'].mean()]"

AI: 이 코드는 '평균보다 매출이 높은 행만 남기기'입니다.

    단계별로 보면:
    1. df['매출'].mean() → 매출 열의 평균값 계산
    2. df['매출'] > 평균 → 각 행이 평균보다 큰지 True/False
    3. df[조건] → True인 행만 선택

    즉, 평균 이하 매출은 제외하고 평균 초과만 남깁니다.
```

### 5. 실무 적용 요청하기
```
사용자: "매일 아침 전날 매출 데이터를 정리해서
       팀장에게 메일 보내야 하는데, 자동화할 수 있어?"

AI: 네, 가능합니다! 다음 순서로 만들어드릴게요:

    1. 매출 데이터 읽기 (pandas)
    2. 필요한 정보 정리 (요약 통계)
    3. 이메일 내용 작성
    4. Outlook으로 발송 (win32com)
    5. 매일 아침 자동 실행 (작업 스케줄러)

    하나씩 만들어볼까요?
```

## 📝 실습 예제: AI와 함께 문제 풀기

아래 코드를 실행해보고, AI에게 질문하며 이해해보세요:

```python
# 실습: 성적 처리 프로그램
scores = [85, 92, 78, 96, 88, 73, 91]

# 1. 평균 계산
average = sum(scores) / len(scores)
print(f"평균: {average:.1f}점")

# 2. 최고점과 최저점
print(f"최고점: {max(scores)}점")
print(f"최저점: {min(scores)}점")

# 3. 80점 이상인 학생 수
passed = [s for s in scores if s >= 80]
print(f"80점 이상: {len(passed)}명")
```

**AI에게 물어볼 것들:**
- "f-string에서 :.1f가 뭔 뜻이야?"
- "리스트 컴프리헨션 없이 같은 결과 내는 코드는?"
- "이 코드에 학생 이름도 추가하려면?"
